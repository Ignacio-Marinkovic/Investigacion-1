Este proyecto utiliza código de terceros para la estimación de las métricas de tensión cosmológica:

Tensiometer: 
Desarrollado por Marco Raveri [arXiv:2105.03324] 
https://github.com/mraveri/tensiometer

Multimeter:
Basado en las herramientas de Leizerovich, Landau y Scóccola para métricas de tensión generalizadas en múltiples datasets.

“Generalized tension metrics for multiple cosmological datasets”
M. Leizerovich, S. Landau, C. Scóccola (2025) 
arXiv:2512.06086 (2025).
